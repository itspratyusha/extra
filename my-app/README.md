# User Dashboard (React)

A simple React app that fetches users from an API and displays their **name, email, and company**.
 Includes **loading, error handling, and search by name**.

## Features
- Fetch users from https://jsonplaceholder.typicode.com/users  
- Displayed name, email and company  
- Loading states
- Search/filter users by name  

## Run Locally
```bash
npm install
npm run dev
